Smart Environmental Monitoring System

A modular system utilizing various sensors to monitor environmental factors like air quality, radiation levels, light intensity, and temperature, with real-time data visualization through a user-friendly interface.

View Live Demo

Project Overview

The Smart Environmental Monitoring System (SEMS) is a comprehensive solution designed for healthcare applications. It features a 3D-printed cubical housing with strategically placed sensors for monitoring various environmental factors. The system is particularly useful in operation rooms, biotech labs, dentistry clinics, and medical imaging centers.

The project combines hardware and software components, with a Raspberry Pi serving as the central processing unit. The system provides real-time monitoring and analysis of environmental conditions, helping maintain optimal and safe environments for healthcare applications.
